# HOW TO PUT THIS ON GITHUB
## Keep It Simple — 5 Steps

---

### STEP 1 — Go to GitHub
Open: **github.com/Grokipaedia**  
Make sure you are logged in.

---

### STEP 2 — Create the repo
Click the **+** button (top right) → **New repository**

Fill in:
- **Repository name:** `iba-managed-agent-guard`
- **Description:** `The harness is not the gate. Pre-execution authorization for Claude Managed Agents.`
- Set to **Public**
- Tick **Add a README file**
- Click **Create repository**

---

### STEP 3 — Open the README file
Inside the new repo, click on **README.md**  
Click the **pencil icon** (Edit this file) — top right of the file

---

### STEP 4 — Paste the README
Select all the existing text in the editor (Ctrl+A)  
Delete it  
Paste the README content Claude gave you  
Scroll down → click **Commit changes**  
Leave the commit message as default → click **Commit changes** again

---

### STEP 5 — Done
Your repo is live at:  
**github.com/Grokipaedia/iba-managed-agent-guard**
